## العربية

**NovaStore** هو نموذج متطور لمتجر إلكتروني متخصص في التسويق بالعمولة، تم بناؤه باستخدام أحدث تقنيات الويب. يركز المشروع بشكل أساسي على تقديم تجربة مستخدم فاخرة مع الالتزام الكامل بمعايير **سهولة الوصول (Accessibility)** لضمان تجربة تصفح مثالية للجميع، بما في ذلك المستخدمين المكفوفين.

### 🌟 المميزات الرئيسية
*   **تجربة مستخدم فاخرة:** تصميم عصري وجذاب باستخدام Tailwind CSS.
*   **دعم كامل لسهولة الوصول (A11y):** 
    *   استخدام بنية HTML دلالية (Semantic HTML).
    *   دعم كامل لقارئات الشاشة عبر علامات ARIA.
    *   رابط "القفز للمحتوى" (Skip to content) لتسهيل التنقل.
*   **أداء فائق:** مبني باستخدام Next.js 14 لضمان سرعة تحميل استثنائية.
*   **دعم اللغة العربية (RTL):** تصميم متوافق تماماً مع اتجاه القراءة من اليمين إلى اليسار.
*   **متجاوب بالكامل:** يعمل بسلاسة على كافة أحجام الشاشات.

### 🛠️ التقنيات المستخدمة
*   **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
*   **Language:** [TypeScript](https://www.typescriptlang.org/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **Icons:** [Lucide React](https://lucide.dev/)
*   **State Management:** React Hooks

### 🚀 تشغيل المشروع محلياً
1. قم بتحميل المستودع:
   ```bash
   git clone [your-repository-link]
   ```
2. انتقل إلى مجلد المشروع:
   ```bash

3. قم بتثبيت المكتبات:

4. ابدأ تشغيل خادم التطوير:

   ```

## English

**NovaStore** is a high-end affiliate marketing e-commerce prototype built with the latest web technologies. The project focuses on delivering a luxury user experience while strictly adhering to **Accessibility (A11y)** standards, ensuring a seamless experience for everyone, including visually impaired users.

### 🌟 Key Features
*   **Luxury UI/UX:** Modern and elegant design using Tailwind CSS.
*   **Full Accessibility Support:**
    *   Semantic HTML structure.
    *   Full screen reader support via ARIA attributes.
    *   "Skip to main content" link for better navigation.
*   **High Performance:** Built with Next.js 14 for lightning-fast loading speeds.
*   **RTL Support:** Fully optimized for Arabic and right-to-left layouts.
*   **Fully Responsive:** Works perfectly on all screen sizes.

### 🛠️ Tech Stack
*   **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
*   **Language:** [TypeScript](https://www.typescriptlang.org/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **Icons:** [Lucide React](https://lucide.dev/)

### 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone [your-repository-link]
   ```
2. Navigate to the project folder:
   ```bash
   cd affiliate-store
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
