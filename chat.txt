أفضل ممارسة (Best Practice) لمشكلة Skip Link مع Sticky Header:

❌ الحل العادي:
#categories-section {
    scroll-margin-top: 73px;
}
- يعمل فقط على عنصر واحد
- يحتاج تكرار لكل قسم مستهدف

✅ الحل الأفضل (Best Practice):
html {
    scroll-padding-top: 73px;
}
@media (min-width: 768px) {
    html {
        scroll-padding-top: 80px;
    }
}

المزايا:
1. يعمل على جميع روابط الإرساء (anchor links) في الموقع دفعة واحدة
2. أسهل للصيانة - تغيير واحد في مكان واحد
3. يغطي أي قسم جديد يتم إضافته في المستقبل تلقائياً
